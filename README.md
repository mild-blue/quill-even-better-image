# Quill even better image

## How to use

1. Register the module

```
Quill.register("modules/imageEmbedOnPaste", QuillImageEmbedOnPaste);
```

2. Enable module in editor options

```
new Quill("#editor-wrapper", {
    modules: {
        imageEmbedOnPaste: true,
    },
});
```
